# README #


### Summary ###

* Provider API for MusicSphere project `http://music-sphere.com`
* Version: 1.0


### How do I get set up? ###

* $> virtualenv env
* source env/bin/activate
* pip install -r requirements.txt
* $> python manage.py syncdb
* $> python manage.py runserver
